## Tech Healt

<h3>Iniciar projeto local</h3>

<h4>Front-end</h4>
instale os pacotes:

```bash
npm install
```
<br>
Execute o front-end:

```bash
npm run dev
```
<hr>

<h4>Back-end</h4>
Acesse /server e instale os pacotes:

```bash
npm install
```
<br>
Execute o back-end em /server:

```bash
npm run dev
```






Acesse a aplicação:
[http://localhost:3000](http://localhost:3000)
